合并两个有序链表

发表于 2019-03-15 | 分类于 算法

题目：将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

答案：

class Solution：
    def mergeTwoLists(self,l1,l2):
     ans = ListNode(0) #创建新链表，最终存放排好序的链表
     p = ans # 让p指向ans链表
     while l1 and l2: #循环，直到l1,l2其中一个为空
        if l1.var <= l2.var:
           p.next = l1
           l1 = l1.next
           p = p.next
        else:
           p.next = l2
           l2 = l2.next
           p = p.next
     #循环结束，必定有一个为空，空即False
     p.next = l1 or l2
     return ans.next
方法的调用和动态约束
删除排序数组中的重复项，返回最终数组长度
