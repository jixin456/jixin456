内置函数isinstance

发表于 2019-03-13 | 分类于 python
  • isinstance函数，专门用于检查类和对象的关系
  • 表达式isinstance(object,cls)检查对象object是否是类cls的实例
  • 当object的类是cls时，得到True,否则得到False
  • 实际上，isinstance可以用于检测任何对象与任何类型的关系，例如一个变量和参数是否为int类型或float类型等
